## Scripts Overview
### Script 1: System Identity Report
Displays system information such as Linux distribution, kernel version, current user, uptime, and system date.

### Script 2: FOSS Package Inspector
Checks whether Git is installed, shows version, license, and prints a short description using case statement.

### Script 3: Disk and Permission Auditor
Audits important system directories, showing permissions and disk usage.

### Script 4: Log File Analyzer
Reads a log file, counts occurrences of a keyword (default: error), and prints summary.

### Script 5: Open Source Manifesto Generator
Generates a personalized open-source philosophy statement based on user input.

## How to Run Scripts

Make scripts executable:
chmod +x script1.sh
chmod +x script2.sh
chmod +x script3.sh
chmod +x script4.sh
chmod +x script5.sh

Run scripts:
./script1.sh
./script2.sh
./script3.sh
./script4.sh /var/log/syslog
./script5.sh
